# Sound Generator

This repo contains a sound generator which can be used to create complex sounds which have multiple layers to the music.

The supported type of sounds are
* Sine wave
* Saw wave (making sounds similar to those of retro video games)
* Square wave (Slightly more refined version of Saw wave)

The dictionary needed to be passed is also in the sound_generator.py file, refer its structure so as to make the sound you desire.

There are infinite possibilities, the only limit is our minds!!
